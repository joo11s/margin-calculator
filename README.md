# 마진계산기 v2

e커머스 플랫폼별 마진을 한눈에 계산할 수 있는 Excel 기반 마진 계산기입니다.

## 주요 기능

| 섹션 | 내용 |
|------|------|
| **섹션1 원가 입력** | 제조원가·포장재 6종·인건비·기타비용 입력, 부가세 자동 구분(과세/면세) |
| **섹션2 배송비** | 무료배송 기준금액·배송비 단가 설정 |
| **섹션3 플랫폼별 마진** | 스마트스토어·쿠팡·카페24·토스쇼핑·기타 5개 플랫폼 동시 비교 |
| **섹션4 역산** | 목표 마진율 입력 → 최소 판매가 자동 산출 |
| **섹션5 BEP** | 월 고정비 기준 손익분기점(수량·매출) 자동 계산 |

## 파일 구조

```
마진계산기.xlsx
├── 시트1: 마진계산기   ← 실제 계산 작업 시트
└── 시트2: 수수료기준표 ← 2026년 6월 기준 플랫폼별 수수료 참고표
```

## 사용법

1. `마진계산기.xlsx` 파일을 열어 **노란색 셀**에 값을 입력합니다.
2. 나머지 계산 결과는 자동으로 채워집니다.

### 색상 안내

| 색상 | 의미 |
|------|------|
| 노란색 (FFFF00) | 사용자 직접 입력 항목 |
| 초록색 (92D050) | 자동 계산 결과 |
| 주황색 (FF6600) | 마진·마진율 |
| 파란색 (00B0F0) | 역산·BEP 결과 |
| 남색 헤더 (1F4E79) | 섹션 헤더 |

## 요구사항

- Microsoft Excel 2016 이상 또는 LibreOffice Calc 7.0 이상
- Python 환경에서 재생성 시: `pip install openpyxl`

## 재생성

```bash
python3 create_excel.py
```
